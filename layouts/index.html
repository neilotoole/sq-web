SQL Sources

sq supports input from and output to common databases.

  • Postgres
  • SQLite
  • MySQL, MariaDB
  • SQL Server, Azure SQL Edge
  • ClickHouse

Other Sources

In addition to SQL sources, sq can import data from these formats:

  • Excel XLSX
  • CSV
  • TSV
  • JSON
  • JSONA (JSON Array)
  • JSONL (JSON Lines)

Output Formats

sq can output to a plethora of formats, or insert results directly to a database table.

  • INSERT to SQL databases
  • Excel XLSX
  • CSV
  • TSV
  • JSON
  • JSONL (JSON Lines)
  • JSONA (JSON Array)
  • XML
  • HTML
  • Markdown
  • Raw (bytes)
